# Prior ML Knowledge: Fluent in Core Algorithm Toolkit

User is an AI/ML Engineer with solid command of the standard ML toolkit: logistic and linear regression, SVMs, decision trees, ensemble methods (Random Forest, XGBoost), deep learning, and transformers/Generative AI. This means lessons should skip algorithm mechanics and go straight to how these techniques are *adapted* for financial use cases, what makes them fit-for-purpose, and what regulatory constraints change the development and deployment pattern.

**Implications:** Never re-explain gradient boosting or backpropagation. Lead with the *financial context* — what the model is predicting, why it matters to the balance sheet, and what the regulator requires.
